# CampusLife - 校园生活交友平台

基于 HTML5 + CSS3 + JavaScript 实现的校园社交互动平台项目

## 技术实现

- **HTML5**：语义化标签，结构清晰
- **CSS3**：响应式布局，支持主题切换
- **JavaScript (ES5)**：模块化设计，功能分离

## 项目结构

```
web-final-project/
├── index.html          # 首页（动态流）
├── login.html          # 登录页
├── register.html       # 注册页
├── profile.html        # 个人中心
├── detail.html         # 动态详情页
├── admin.html          # 管理员后台
├── css/                # 样式文件
│   ├── reset.css       # 样式重置
│   ├── variables.css   # CSS 变量
│   ├── components.css  # 组件样式
│   └── layout.css      # 布局样式
├── js/                 # JavaScript 文件
│   ├── data-store.js   # 数据存储层
│   ├── auth.js         # 认证授权
│   ├── render.js       # 渲染函数
│   ├── main.js         # 主业务逻辑
│   ├── profile.js      # 个人中心逻辑
│   └── admin.js        # 管理员逻辑
└── img/                # 图片资源
```

## 运行方式

1. 直接使用浏览器打开 `index.html` 即可运行
2. 推荐使用 Chrome、Firefox、Edge 等现代浏览器

## 默认账号

- **管理员账号**：学号 `admin`，密码 `123456`
- **普通用户**：可在注册页面创建新账号

## 数据存储

- 使用浏览器 localStorage 存储所有数据
- 清除浏览器数据会丢失所有信息
- 无需后端服务器即可运行

## 主要功能演示

### 注册登录
- 使用学号注册，支持设置头像、昵称、兴趣标签
- 登录支持"记住我"、"密码可见"功能

### 发布动态
- 支持文字、图片（URL链接）、话题标签
- 可设置可见范围（公开/仅好友）
- 自动识别内容中的 `#标签` 格式

### 浏览互动
- 首页支持"推荐"和"关注"两种模式
- 支持点赞、评论、表情回复
- 动态详情页查看完整内容和评论列表

### 个人中心
- 查看个人资料、动态历史
- 查看粉丝/关注列表
- 编辑个人信息

### 管理员后台
- 用户管理：查看用户列表、封禁/解封用户、重置头像
- 内容管理：查看所有动态、删除不当内容
- 数据统计：用户数量、动态数量统计

### 其他功能
- 主题切换（日间/夜间模式）
- 搜索功能（搜索动态和话题）
- 热门话题榜（自动统计）
- 活跃用户推荐（根据发帖量）
